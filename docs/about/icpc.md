# ICPC 简介

**ACM-ICPC** 是由美国计算机协会（ACM）主办的世界上规模最大、历史最悠久的大学生程序设计竞赛，被誉为「**程序设计界的奥林匹克**」。

---

## 赛事特点

| 特点 | 说明 |
|------|------|
| **团队作战** | 3 人一组，共用一台电脑，在 5 小时内解决 8-13 道算法题 |
| **高强度** | 考察算法设计、代码实现、团队协作和临场应变能力 |
| **全球参与** | 每年吸引来自全球 3000+ 所大学的顶尖选手参赛 |

---

## 赛制流程

```
校赛 → 区域赛（亚洲区） → World Finals（全球总决赛）
```

- **校赛**：各高校自行组织，选拔优秀选手
- **区域赛**：每年秋季举行，亚洲设多个赛区（西安、北京、上海等）
- **全球总决赛**：区域赛优胜队伍晋级，每年春季举行

---

## 对我们协会的意义

ICPC 是协会的核心目标赛事。协会的训练体系围绕 ICPC 的要求设计，从基础算法到高级数据结构，系统化培养队员的竞赛能力。
