# 什么是算法竞赛和 ACM

<div class="about-section">

## 🎯 算法竞赛概述

**算法竞赛**（也称程序设计竞赛）是一种以编程解决算法问题为核心的智力竞赛。参赛者需要在限定时间内，设计算法并编写代码来解决一系列具有挑战性的问题。

**ACM**（Association for Computing Machinery，美国计算机协会）是全球最大的计算机科学学术组织。ACM 主办的 **ICPC** 是算法竞赛中最具影响力的赛事，因此在国内，「ACM 竞赛」常被用作算法竞赛的统称。

| 赛事 | 主办方 | 形式 | 难度 | 地位 |
|------|--------|------|------|------|
| **ICPC** | ACM（美国） | 3 人团队 | ⭐⭐⭐⭐⭐ | 世界顶级，算法竞赛的「奥林匹克」 |
| **CCPC** | CCF（中国） | 3 人团队 | ⭐⭐⭐⭐⭐ | 国内顶级，与 ICPC 齐名 |
| **蓝桥杯** | 工信部（中国） | 个人 | ⭐⭐~⭐⭐⭐ | 覆盖面最广，适合入门到进阶 |
| **天梯赛** | 教育部教指委（中国） | 10 人团队积分 | ⭐⭐~⭐⭐⭐⭐ | 强调团队积分，鼓励全员参与 |

### 它们之间的关系

```
┌─────────────────────────────────────────┐
│              算法竞赛生态                │
│                                          │
│   ICPC ──────── 世界顶级 ────── CCPC     │
│    │              ↑              │       │
│    │         天梯赛              │       │
│    │         (全员练兵)           │       │
│    ↓              │              ↓       │
│    └────── 蓝桥杯 (个人入门) ────┘       │
│                                          │
│   难度梯度：蓝桥杯 → 天梯赛 → ICPC/CCPC  │
│   形式互补：个人赛 + 团队赛 + 积分赛     │
└─────────────────────────────────────────┘
```

- **蓝桥杯**是入门首选，个人参赛，难度适中，适合积累比赛经验
- **天梯赛**是团体练兵，鼓励全协会参与，检验整体训练水平
- **ICPC** 和 **CCPC** 是终极目标，团队作战，代表协会最高水平

## ICPC — 国际大学生程序设计竞赛

  <div>

- 🌍 全球 3000+ 大学参赛
- 👥 3 人一组，5 小时解决 8-13 题
- 📅 每年秋季区域赛 → 次年春季全球总决赛
- 🎯 协会的**最高目标赛事**
