## 第一步：掌握编程语言以及基础数据结构的用法

算法竞赛主要使用 **C/C++**，你需要先掌握：

- 输入输出（cin/cout、scanf/printf）
- C/C++ 基本语法（变量、循环、分支、函数、数组、引用）
- STL 标准库（string、vector、map、set、queue、stack 等）

协会预计每年 9-10 月有系统的 C++ 语法培训。

## 第三步：系统学习算法、熟练运用

### 推荐学习路线

| 阶段 | 主题 | 核心内容 |
|------|------|----------|
| 1 | 语言基础 | C/C++ 语法巩固、STL 熟练运用、基础调试技巧 |
| 2 | 模拟与枚举 | 理解题意 → 模拟过程 → 验证结果，培养翻译能力 |
| 3 | 贪心与排序 | 局部最优推全局最优，掌握常见贪心策略与证明思路 |
| 4 | 搜索 | DFS（深度优先）、BFS（广度优先）、回溯剪枝、记忆化 |
| 5 | 基础数据结构 | 并查集、前缀和/差分、单调栈/队列、树状数组、线段树入门 |
| 6 | 动态规划 | 线性 DP、背包、区间 DP、状压 DP、树形 DP 入门 |
| 7 | 图论 | 最短路（Dijkstra/Floyd）、最小生成树、拓扑排序、连通性 |
| 8 | 数论与组合数学 | 素数筛、快速幂、GCD/LCM、排列组合、容斥原理 |
| 9 | 字符串 | KMP、Trie 树、字符串哈希、Manacher 入门 |
| ★ | 高级专题（进阶） | 网络流、计算几何、AC 自动机、主席树、莫队算法 |

**推荐资料：**

- 《算法竞赛入门经典》（刘汝佳）— ICPC 必备教材
- [OI Wiki](https://oi-wiki.org) — 免费在线竞赛知识库
- [AcWing](https://www.acwing.com) — 适合新手入门的算法课程和算法模板
- [左程云算法](https://space.bilibili.com/8888480?spm_id_from=333.337.0.0) — 非常详细的算法教学
- [科学刷题指南](https://leetcode.cn/circle/discuss/RvFUtj/) — 如何科学刷题？

## 常见问题

<details class="faq-item">
<summary class="faq-question">零基础能跟上吗？</summary>
<div class="faq-answer">
完全可以！大部分同学刚加入时都是零基础。刚进入协会时会有系统的 C/C++ 语言入门培训，从 Hello World 开始讲起。只要你愿意花时间学习，一定能跟上。
</div>
</details>

<details class="faq-item">
<summary class="faq-question">每周需要投入多少时间？</summary>
<div class="faq-answer">

- **入门阶段**：每周 3-8 小时（培训 + 做题）
- **进阶阶段**：每周 15-20 小时（专题学习 + 练习）
- **竞赛冲刺**：每周 35+ 小时（高强度刷题 + 组队训练）

协会不强制要求，鼓励按照自己的节奏来安排。

</div>
</details>

<details class="faq-item">
<summary class="faq-question">非计算机专业可以参加吗？</summary>
<div class="faq-answer">
当然！ICPC 对参赛选手的专业没有限制。很多顶尖选手来自数学、物理等非 CS 专业。关键是你的兴趣与坚持，而不是你的专业。
</div>
</details>
