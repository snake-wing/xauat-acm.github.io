# 入门指南

<div class="about-section">

欢迎加入 XAUAT ACM 协会！这份指南将帮助你快速入门算法竞赛。

---

## 第一步：掌握编程语言

算法竞赛主要使用 **C/C++**，你需要先掌握：

- 基本语法（变量、循环、分支、函数）
- 数组、字符串操作
- 指针与引用
- **STL 标准库**（vector、map、set、queue、stack 等）
- 输入输出（cin/cout、scanf/printf）

协会每年 10-11 月有系统的 C 语言培训，从零开始学完全没问题。

## 第三步：系统学习算法

推荐学习路线：

```
语言基础 → 模拟/枚举 → 贪心 → 搜索(DFS/BFS) →
动态规划 → 基础数据结构 → 图论 → 高级专题
```

**推荐资料：**
- 《算法竞赛入门经典》（刘汝佳）— ICPC 必备教材
- [OI Wiki](https://oi-wiki.org) — 免费在线竞赛知识库
- [AcWing](https://www.acwing.com) — 高质量算法课程

## 常见问题

<details class="faq-item">
<summary class="faq-question">零基础能跟上吗？</summary>
<div class="faq-answer">
完全可以！大部分同学刚加入时都是零基础。协会每年秋季有系统的 C 语言入门培训，从 Hello World 开始讲起。只要你愿意花时间学习，一定能跟上。
</div>
</details>

<details class="faq-item">
<summary class="faq-question">每周需要投入多少时间？</summary>
<div class="faq-answer">

- **入门阶段**：每周 3-5 小时（培训 + 做题）
- **进阶阶段**：每周 8-12 小时（专题学习 + 练习）
- **竞赛冲刺**：每周 15+ 小时（高强度刷题 + 组队训练）

协会不强制要求，鼓励按照自己的节奏来安排。

</div>
</details>

<details class="faq-item">
<summary class="faq-question">非计算机专业可以参加吗？</summary>
<div class="faq-answer">
当然！ICPC 对参赛选手的专业没有限制。很多顶尖选手来自数学、物理等非 CS 专业。关键是你的算法和编程水平，不是你的专业。
</div>
</details>
