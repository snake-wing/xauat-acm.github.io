# 蓝桥杯简介

  </div>

**蓝桥杯** 是由工业和信息化部人才交流中心主办的全国性 IT 学科赛事，覆盖范围广、参与人数多，是国内高校**参与度最高**的程序设计竞赛之一。

---

## 🏅 赛事特点

| 特点 | 说明 |
|------|------|
| 📊 **覆盖面广** | 包括 C/C++、Java、Python 等多个组别 |
| 📈 **难度分级** | 省赛 → 国赛，层次分明，适合各阶段同学 |
| 🎓 **认可度高** | 获奖在保研、奖学金评定中是重要加分项 |
| 🏫 **独立参赛** | 个人赛形式，不依赖团队配合 |

---

## 📋 赛制

```
省赛（每年 4 月） → 国赛（每年 5-6 月）
```

- **省赛**：各省分别举办，选拔国赛选手
- **国赛**：省赛一等奖选手晋级，在北京统一举行

---

## 🎯 对我们协会的意义

蓝桥杯是协会**入门首选赛事**。相比 ICPC/CCPC 的团队赛和高难度，蓝桥杯更适合大一、大二同学积累比赛经验。
