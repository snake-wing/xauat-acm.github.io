# CCPC 简介

**CCPC** 是由中国计算机学会（CCF）主办的国内顶级大学生程序设计竞赛，与 ICPC 齐名，是国内算法竞赛的**两大赛事**之一。

---

## 赛事特点

| 特点 | 说明 |
|------|------|
| **中国标准** | 由中国计算机学会主办，更贴近国内高校竞赛需求 |
| **分站赛制** | 在全国多地设有分站赛（秦皇岛、威海、哈尔滨、深圳等） |
| **总决赛** | 各分站优胜队伍晋级年度总决赛 |

---

## 与 ICPC 的关系

CCPC 和 ICPC 是两个独立赛事，但：

- 考察内容高度重合（算法 + 数据结构 + 数学）
- 训练体系可以共享
- 很多队伍同时参加两项赛事
- CCPC 的比赛时间通常在 ICPC 区域赛之前，是重要的热身机会

---

## 对我们协会的意义

CCPC 是协会重点参加的赛事，与 ICPC 并列为我们训练体系的**双核心目标**。
