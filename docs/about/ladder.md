# 天梯赛简介

**天梯赛** 是由教育部高等学校计算机类专业教学指导委员会等机构联合主办的全国性赛事，以**团队积分制**为核心特色。

---

## 赛事特点

| 特点 | 说明 |
|------|------|
| **梯级积分** | 题目按难度分 L1（基础）、L2（进阶）、L3（挑战），不同难度分值不同 |
| **团队积分** | 每校派出 10 人队伍，累计个人得分作为团队总分 |
| **全校参与** | 不限专业和年级，鼓励更多同学参与 |

---

## 赛制

- **形式**：线上统一时间比赛，3 小时完成
- **题目**：L1（8题）→ L2（4题）→ L3（3题）
- **评分**：团队总分 = 所有队员得分之和

---

## 对我们协会的意义

天梯赛是检验**整体训练成果**的好机会。不同于 ICPC 的精英模式，天梯赛鼓励全员参与，有助于带动协会整体水平的提升。
