# 关于我们

<div class="about-section">

## 🏫 西安建筑科技大学 ACM 协会

西安建筑科技大学 ACM 协会是一个面向全校学生的学术科技类社团，致力于培养算法设计与编程能力，组织参加 ACM-ICPC 国际大学生程序设计竞赛、CCPC 中国大学生程序设计竞赛等赛事。

---

## 🎯 主要活动

| 学期 | 活动 |
|------|------|
| **第一学期** | 百团纳新、新生交流会、C/C++ 零基础培训、算法讲堂、迎新杯程序设计大赛 |
| **第二学期** | 进阶算法课程、校赛选拔、暑期集训 |

### 算法讲堂

涵盖内容从基础到进阶：

- **基础专题**：贪心算法、搜索、动态规划基础、背包问题
- **进阶专题**：数据结构（线段树/树状数组）、数论、图论、字符串算法
