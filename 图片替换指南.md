# 网站图片替换指南

本文档说明如何为网站准备和替换所有图片资源。

---

## 图片清单与命名规范

所有图片放在 `docs/public/images/` 目录下。

> ⚠️ 当前为 1×1 像素占位图，需替换为实际图片

| 图片用途 | 文件名 | 推荐尺寸 | 格式 | 说明 |
|----------|--------|----------|------|------|
| 网站图标 | `favicon.png` | 64×64 px | PNG | 浏览器标签页图标 |
| ICPC 竞赛 Logo | `icpc-logo.png` | 400×160 px | PNG | ICPC 官方标识 |
| CCPC 竞赛 Logo | `ccpc-logo.png` | 400×160 px | PNG | CCPC 官方标识 |
| 蓝桥杯 Logo | `lanqiao-logo.png` | 400×160 px | PNG | 蓝桥杯官方标识 |
| 天梯赛 Logo | `ladder-logo.png` | 400×160 px | PNG | 天梯赛官方标识 |

---

## 替换步骤

### 1. 准备图片

将官方 Logo 处理为推荐尺寸的 PNG 图片，确保：
- 背景透明或与页面背景协调
- 清晰无锯齿
- 文件大小控制在 50KB 以内

### 2. 放入图片目录

```bash
# 将准备好的图片复制到 docs/public/images/ 目录
docs/public/images/
  ├── favicon.png        # 网站图标
  ├── icpc-logo.png      # ICPC Logo
  ├── ccpc-logo.png      # CCPC Logo
  ├── lanqiao-logo.png   # 蓝桥杯 Logo
  ├── ladder-logo.png    # 天梯赛 Logo
  ├── hero-bg.jpg        # 首页 Banner 背景图（可选）
  └── gallery/           # 获奖瞬间照片（可选）
```

### 3. 获取官方 Logo 的渠道

| 比赛 | Logo 获取方式 |
|------|--------------|
| **ICPC** | 访问 https://icpc.global 页脚或媒体资源区 |
| **CCPC** | 访问 https://ccpc.io 页头 Logo 区 |
| **蓝桥杯** | 访问 https://design.lanqiao.cn 悬浮窗口下载 |
| **天梯赛** | 访问 https://gplt.patest.cn 或使用赛事相关素材 |

### 4. 验证

替换完成后运行 `npm run dev`，确认所有图片正常显示。

## 关于社交图标

顶部导航栏的 GitHub、QQ、Bilibili 图标使用的是内联 SVG 格式（写在代码里），这是 VitePress 框架的要求，**不需要替换为图片**。

如需更换这些图标，编辑 `docs/.vitepress/config.mts` 中 `socialLinks` 部分的 `svg` 字段。

---

## 首页 Banner 背景图

首页顶部的大图 Banner 位于 `docs/public/images/hero-bg.jpg`。

放入图片后，编辑 `docs/.vitepress/theme/components/HeroBanner.vue`，将：

```html
<!-- 替换为实际图片：<img src="/images/hero-bg.jpg" ... /> -->
<div class="hero-bg-fallback">...</div>
```

改为：

```html
<img src="/images/hero-bg.jpg" alt="XAUAT ACM Banner" class="hero-bg-img" />
```
